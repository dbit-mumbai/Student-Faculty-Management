# Student-Faculty-Management
You can manage students project related details using this software.
To run this project
1. Download the project files
2. Run Xampp
3. Copy the contents of the files into the htdocs folder of Xampp
4. Run Apache Server and Mysql from Xampp
5. Open web browser and enter the URL: localhost to ensure that apache is started
6. Open the phpmyadmin console by entering URL: localhost/phpmyadmin
7. enter the username and password what you have set
8. Import the project_det.sql file in phpmyadmin console
9. In the URL Enter: localhost/PHP/
The project is now wokring and You have to configure the connect.php incase the output doesnot connect to database as you
have to enter the credentials of the phpmyadmin username and password for your computer.
